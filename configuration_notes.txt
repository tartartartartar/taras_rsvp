Note on how to install for Taras' machine (Kubuntu 12.10)


Setting up bach to handle all the rvm stuff:

add to .bashrc:
    PATH=$PATH:$HOME/.rvm/bin # Add RVM to PATH for scripting
    export PATH="/usr/local/heroku/bin:$PATH"
    export rvm_path=$HOME/.rvm
    export rvm_bin_path=$HOME/.rvm/bin
    export rvm_prefix=$HOME
    export GEM_HOME=/home/taras/.rvm/gems/ruby-2.0.0-p195
    export IRBRC=/home/taras/.rvm/rubies/ruby-2.0.0-p195/.irbrc
    export MY_RUBY_HOME=/home/taras/.rvm/rubies/ruby-2.0.0-p195
    export GEM_PATH=/home/taras/.rvm/gems/ruby-2.0.0-p195:/home/taras/.rvm/gems/ruby-2.0.0-p195@global
    export RUBY_VERSION=ruby-2.0.0-p195
    source /home/taras/.rvm/scripts/rvm
